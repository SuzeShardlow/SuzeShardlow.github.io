Icebreaker was the final project I built on the 12-week coding bootcamp I attended.

I wrote it in seven days, using AngularJS on Rails and internal and external APIs.

  • Launch app
  • View code

This full stack app is for techies who know they have to attend networking events but are too scared to go on their own.

The app pulls event data from the Meetup.com API, from which the user can select an event. The details from the Meetup.com API are then populated into the internal API as a pre-meet. Other users can then join that pre-meet so that they know at least one person at the main event.

Skedaddle is a MEN (MongoDB, Express, Node.js) stack app I built which draws data from the TfL API to help Tube travellers.

I created this app in seven days, during my 12-week coding bootcamp.

  • Launch app
  • View code

The TfL API has thousands of endpoints and I wanted to see if I could build an app which allows users to log in securely, plan journeys and save them to the database, as well as view station information.

I am now creating an Alexa skill which also uses the TfL API.
